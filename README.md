Installationsanleitung

Die folgende Anleitung verwendet die Entwicklungsumgebung PyCharm. Sie können jedoch auch eine andere IDE Ihrer Wahl verwenden.

1. Die neuste Version von Python muss installiert werden, sofern dies noch nicht gegeben ist. Dabei muss beachtet werden, welche Operating System Ihr Computer verwendet. Der Link für den Download: https://www.python.org/downloads/release/python-3132/
2. Laden Sie die Zip-Datei von diesem Projekt herunter. Dafür müssen Sie auf Code > Download Zip klicken.
3. Entpacken Sie die ZIP-Datei in einem Ordner Ihrer Wahl auf Ihrem Computer. Ich empfehle dafür direkt den Desktop, damit Sie den Ordner leicht und schnell finden können.
4. Installieren Sie die Entwicklungsumbegung PyCharm auf Ihrem Computer. Link für den Download: https://www.jetbrains.com/pycharm/download/?section=windows Achten Sie dabei darauf, die kostenlose Community Edition zu installieren.
5. In PyCharm navigieren Sie zu File > Open und dann können Sie zum Projektverzeichnis navigieren, in dem Sie das Projekt entpackt haben und dieses öffnen.
6. Sie werden nun aufgefordert dem Projekt zu vertrauen und können dann das Projekt im selben oder in einem neuen Fenster öffnen lassen.
7. Um die benötigten Python-Bibliotheken zu installieren, können sie im Terminal den Befehl "pip install -r requirements.txt" verwenden. Dafür sollten Sie eine virtuelle Python-Umgebung einrchten. Link für die Einrichtung in PyCharm: https://www.jetbrains.com/help/pycharm/creating-virtual-environment.html#python_create_virtual_env
8. Navigieren Sie zu LoginView.py auf der linken Seite.
9. Um das Projekt nun auszuführen, klicken Sie oben rechts auf den Run-Button. Die Bedienoberfläche des Projekts sollte sich nun in einem seperaten Fenster öffnen.
